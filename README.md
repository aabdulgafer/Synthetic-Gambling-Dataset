# Synthetic Gambling Conversations Dataset — Publication Package (v1.0.0)

This package contains **monologues** and **multi-user threads** generated by a controlled pipeline for gambling-related social media simulation.

## Files
- `monologues_with_flags.csv`: All monologues with content/PII flags and duplicate indicators.
- `threads_with_flags.csv`: All threads with content/PII flags and duplicate indicators.
- `monologues_dedup.csv`: Monologues with normalized-text deduplication applied (keep-first).
- `threads_dedup.csv`: Threads with normalized-text deduplication applied (keep-first).
- `monologues_validation_report.csv`: Per-row validation details (full) — previously generated.
- `threads_validation_report.csv`: Per-row validation details (full) — previously generated.
- `DATACARD.md`: Detailed factsheet.


## Schema (added columns)
- `normalized_text` (string; for transparency)
- `is_duplicate_strict` (bool)
- `is_duplicate_normalized` (bool)
- `contains_email`, `contains_phone`, `contains_creditcard_like`, `contains_iban_like` (bool)
- `contains_profanity` (bool)
- `mentions_money` (bool)
- `contains_url` (bool)

## Recommended usage
- Prefer the `*_deduplication.csv` for training.
- If your use case requires family-safe text, filter `contains_profanity == False`.
- If you study financial risk behaviors, filter on `mentions_money == True`.
-  mFor most research use cases, we recommend using `*_deduplication.csv`.
